CSV Format Instructions
Required CSV Format:

Your CSV file should have 2 columns without headers:

item_id,quantity_sold
ABC123,15
XYZ456,8
DEF789,25
Important Notes:
  • Column 1: Item ObjectID (from Algolia)
  • Column 2: Quantity sold
  • No headers in the CSV file
  • Stock will be reduced by the quantity sold
  • Daily sales counter will be updated
  • Stock cannot go below 0
Important: Make sure all item IDs in your CSV exist in the inventory. Invalid IDs will be skipped.

How It Works
  1. Upload CSV: Your sales data is read from the CSV file
  2. Validation: System checks if all item IDs exist in inventory
  3. Stock Update: Current stock is reduced by quantity sold
  4. Sales Counter: Daily sales metric is incremented
  5. Redirect: You'll be redirected to the inventory page to see updates
